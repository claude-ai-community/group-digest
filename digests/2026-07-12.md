## 本期看点

Austin 的 [PM 文章](https://austinxyz.github.io/blogs/blog/pm-ai-era)([回顾](digests/2026-07-11.md))成了跳板,烧出一圈延伸讨论——"Agent 会越来越薄"的预判几乎立刻被现场纠偏(大代码库里 agentic search 其实和 LSP 混用),印度外包的 AI backup 角色被判定短多长空,王煜全抛出 2029 年泡沫破裂、"行为数字化"、"服务平权"一套硬核预测,收尾两篇文章分别扒了 Anthropic 和 OpenAI 各自"没那么干净"的商业化路径。

## 讨论

- **Agent 会越来越薄?Agentic Search vs RAG 被现场纠偏**:承接 [7/11 的 Claude Code 架构拆解](digests/2026-07-11.md),群里先重申了"agent 会越来越薄,langgraph 这类 graph flow 交给大模型端处理"的判断,随即有群友指出这个不完全正确:Claude Code 用 grep/glob/read 实时探索取代预索引,这个设计在中小代码库确实成立,但 grep 是"找到多少算多少",不保证找全——库大到千万行,或要做穷举型任务(比如改接口找全调用点),就得靠 LSP/调用图这种编译级索引。Claude Code 自己其实也带了 LSP,实际是混合用的。

- **AI 与印度外包:短期利好,长期利空**:Austin 的判断——短期内,让一个组织变成"AI 组织"(只靠 senior 带一堆 AI 完成工作)对人和组织都还没准备好,所以会用印度相对便宜的人力做 AI 的 backup,毕竟美国/硅谷人力太贵。但等到 senior 真能完全带 AI 替代原来的团队,印度的 backup 角色就不再需要了——而"靠印度人直接晋升 senior 去带 AI"这条路本身也不太可能走通。

- **MIT 人脑推理研究:LLM 路线是否走反?**:一项研究指出人脑推理并不依赖语言,这对当前"语言模型即推理"的 LLM 主流路线提出了质疑,杨立昆转发力挺。[原文](https://mp.weixin.qq.com/s/bYKKMgukQoj7Sxk8Ojxg0A)

- **王煜全:2029 年 AI 泡沫、"行为数字化"与"服务平权"**([视频](https://www.youtube.com/watch?v=140Ek7GJJoE),易论 AI 发布):
  - **泡沫周期**:预测 AI 泡沫可能在 2029 年左右破裂,2028 年会先形成类似"郁金香泡沫"的狂热期——技术社会变革的规律相似,人类对技术的预期总是超过实际落地速度,当前正处于从恐慌(怕被替代)转向狂热(发现新机会)的转折点。
  - **从内容数字化到"行为数字化"**:美国的优势在内容生成(AIGC),基于互联网内容积累;中国的机会在移动端的"行为数字化"——记录并聚集海量连续行为数据能产生"大行为模型",商业价值高于内容理解,因为它能预判并直接给出行动(Action),而不只是建议。
  - **以 AI 为核心重构流程**:过去以人为核心构建流程,未来以 Agent 为核心;优秀的业务模式不再是人加班熬夜,而是让勤奋的 Agent 完成具体环节,人站在旁边监控修正。大模型就像当年的电力系统,未来的竞争在于谁能提供最稳定强大的"电力"支撑各环节 Agent。
  - **服务平权**:现在的互联网平台像"数字帝国",把用户锁在围墙内;AI 会赋予个人"文明级认知能力"去对抗平台,未来的颠覆者是站在"用户侧"、完全为用户谋利的 Agent。工业革命实现了物质平权,AI 时代将实现服务平权——把原本只有富人能享受的专家级服务(私人医生、理财顾问、定制购物)大规模低成本提供给普通人。
  - **从"打枪"到"当元帅"**:传统"打工人"模式(由别人定义工作)可能终结,人不再需要学具体执行技能,而要学做选择、做安排、做结果校验;教育应从教人当具体专家转向教人当统筹全局的领导者。人类不会被挤出世界,而是被挤出具体执行环节,人的洞察力与 AI 的系统性结合形成"人机混合认知"。
  - 另提到未来 AI 不应高度集权,而应是类似大脑多系统协作的分布式智能,技术与文化会双向演进。

- **Anthropic 与 OpenAI:谁更"脏"?**(两篇文章,核心诊断对比):
  - 一篇讲"给程序员的一课":写代码已不是最重要的能力;还提到安全后门隐患通报,以及阿里全员卸载 Claude 转用 QCode 的传闻。[原文](https://mp.weixin.qq.com/s/fZjOGewF9PD7VR1k81ernQ)
  - 另一篇《从公园长椅到万亿美元裂缝:Anthropic 的五年背离与 OpenAI 的掠夺底色》([原文](https://mp.weixin.qq.com/s/b8hRpdWWxe9gNGAebiUFww))核心诊断:两家公司的"脏"不在同一方向——Anthropic 是内外失控加双标,OpenAI 是对外掠夺,但根源一致,安全承诺在商业化压力下从硬约束沦为软修辞。Anthropic 从 2021 年 Dario Amodei 带 14 名 OpenAI 核心研究员在公园长椅上规划"安全优先"乌托邦,五年后估值逼近万亿、ARR 突破 300 亿美元,但背负 15 亿美元版权和解、30 亿美元歌词索赔、51 万行代码泄露与"双标"蒸馏指控——因为人设是"安全派 + 宪法 AI + 诚实",每爆一条黑料都比 OpenAI 更违和。OpenAI 则从未装过道德高地:苹果以系统性窃取商业机密为由诉至联邦法院、《纽约时报》申请证据制裁、佛州总检察长提起产品责任诉讼,面临 18 起产品责任诉讼。核心命题:当伦理信用被纳入估值模型,合规已不再是成本中心,而是生死线——两家公司在 IPO 前夜同时面临合规清算。
