## 怎么运作

- 群主(Austin)定期把群里的好讨论整理进来(AI 辅助,人工校审)。
- **欢迎 PR / Issue**:发现遗漏的好工具、想补充心得,直接提。
- 内容按话题归纳,不逐条搬运聊天记录。

## 隐私约定

- **原始聊天记录 / 小微总结只存维护者本地(raw/,已 gitignore),永不上传**;线上只有匿名化后的精华。
- 群友发言**匿名化**处理(不出现微信名/头像/联系方式)。
- 只沉淀"知识"(工具、方法、结论),不沉淀"聊天"。
- 不想被收录的内容,联系群主删除。
